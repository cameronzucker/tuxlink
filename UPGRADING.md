# Upgrading Tuxlink

This document lists upgrade instructions for each MAJOR release. If you are upgrading across a MAJOR boundary (e.g., `v1.x` → `v2.0`), follow the matching section before launching the new AppImage.

MINOR and PATCH upgrades are safe with no special steps — download the new AppImage and run it. The first launch detects the existing config and continues without re-running the wizard.

## v0.0.1

Initial release. No prior version to upgrade from.

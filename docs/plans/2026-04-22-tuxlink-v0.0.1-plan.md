# Tuxlink v0.0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a Tauri-packaged Linux desktop Winlink client that lets a non-Cameron user install tuxlink, enter callsign credentials, send a Winlink telnet message, receive a reply, and never touch Pat directly.

**Architecture:** Single-crate Rust binary, Tauri 2.x shell with a native OS menu bar and system tray. React + TypeScript + Vite frontend. Pat v1.0.0 bundled as a child process managed by tuxlink (spawn on launch, SIGTERM on quit, PID file at `$XDG_RUNTIME_DIR/tuxlink-pat.pid`). Telnet-only protocol surface. No SQLite; Pat's HTTP API is the authoritative mailbox source. Local fake Winlink CMS for integration tests.

**Tech Stack:** Rust 1.75+, Tauri 2, React 18, TypeScript 5, Vite, TanStack Query 5, Radix UI primitives, react-virtuoso, Pat v1.0.0 (Go binary, bundled), `std::process` for child management, `reqwest` (blocking mode via `spawn_blocking`) for Pat HTTP API, integration-test fake CMS written in Rust.

**Deferred to v0.1 (do NOT implement in v0.0.1):** workspace split into multiple crates, `tuxlink-protocol-native`, VARA-via-WINE, hamlib, AX.25 packet, P2P sessions, RMS Relay, scheduled sessions, radio-only mode, position reports, attachments, HTML form rendering (ICS-213), session log translation layer, SQLite, Flatpak/deb/rpm packaging (AppImage only).

```rust
use serde::{Deserialize, Deserializer, Serialize};

pub const CONFIG_SCHEMA_VERSION: u32 = 1;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Config {
    #[serde(deserialize_with = "deserialize_schema_version")]
    pub schema_version: u32,
    #[serde(deserialize_with = "deserialize_nonempty_string")]
    pub callsign: String,
    #[serde(deserialize_with = "deserialize_nonempty_string")]
    pub grid_square: String,
    #[serde(deserialize_with = "deserialize_nonempty_string")]
    pub pat_mbo_address: String,
    pub winlink_password_present: bool,
    pub wizard_completed: bool,
}

fn deserialize_schema_version<'de, D>(d: D) -> Result<u32, D::Error>
where D: Deserializer<'de> {
    let v = u32::deserialize(d)?;
    if v != CONFIG_SCHEMA_VERSION {
        return Err(serde::de::Error::custom(format!(
            "unsupported config schema_version {} (expected {})",
            v, CONFIG_SCHEMA_VERSION
        )));
    }
    Ok(v)
}

fn deserialize_nonempty_string<'de, D>(d: D) -> Result<String, D::Error>
where D: Deserializer<'de> {
    let s = String::deserialize(d)?;
    if s.is_empty() {
        return Err(serde::de::Error::custom("field must not be empty"));
    }
    Ok(s)
}

/// Resolve the config file path. Honors XDG_CONFIG_HOME, falls back to
/// ~/.config/tuxlink/config.json.
pub fn config_path() -> std::path::PathBuf {
    let base = std::env::var_os("XDG_CONFIG_HOME")
        .map(std::path::PathBuf::from)
        .unwrap_or_else(|| {
            let home = std::env::var_os("HOME").expect("HOME must be set");
            std::path::PathBuf::from(home).join(".config")
        });
    base.join("tuxlink").join("config.json")
}
```

### Task 3: Pat child-process lifecycle

**Files:**
- Create: `src-tauri/src/pat_process.rs`
- Create: `src-tauri/tests/pat_process_test.rs`
- Modify: `src-tauri/src/lib.rs` (add `pub mod pat_process;`)

**Behavior:** Tuxlink spawns the bundled `pat` binary as a child process. The child is launched with its HTTP API bound to an ephemeral port on `127.0.0.1` (NEVER `0.0.0.0`). The spawned PID is written to `$XDG_RUNTIME_DIR/tuxlink/pat.pid` (or `/tmp/tuxlink-$USER/pat.pid` if XDG_RUNTIME_DIR is unset). On shutdown, tuxlink sends SIGTERM to the child, waits up to 5 seconds for graceful exit, then SIGKILL if needed. On startup, if a PID file exists and that PID belongs to a still-running `pat` process owned by this user, tuxlink must NOT spawn another — it logs the conflict and surfaces an error to the UI asking the user to quit the other instance.

Pat's CLI flags we rely on: `pat http --listen 127.0.0.1:<port>` and `--mbox <dir>` for the mailbox location, plus `--config <path>` for Pat's config.json (written by the wizard in a later task; this task takes the paths as parameters).

- [ ] **Step 1: Write the failing test — happy path spawn and shutdown**

```rust
use std::path::PathBuf;
use std::time::Duration;
use tuxlink_lib::pat_process::{PatProcess, PatSpawnOptions};

/// This test requires a `pat` binary in PATH or at the path passed in.
/// CI installs Pat at a known location (see Task 19).
fn pat_binary() -> PathBuf {
    std::env::var_os("PAT_BINARY")
        .map(PathBuf::from)
        .unwrap_or_else(|| PathBuf::from("pat"))
}

#[test]
fn test_spawn_and_graceful_shutdown() {
    let tmp = tempfile::tempdir().expect("tmpdir");
    let opts = PatSpawnOptions {
        binary: pat_binary(),
        config_path: tmp.path().join("pat-config.json"),
        mbox_dir: tmp.path().join("mbox"),
        http_listen_port: 0, // 0 = let OS pick an ephemeral port
        pid_file: tmp.path().join("pat.pid"),
    };
    // Write a minimal Pat config so Pat does not prompt on stdin.
    std::fs::write(&opts.config_path, r#"{
        "mycall": "TEST1",
        "secure_login_password": "x",
        "locator": "AA00aa"
    }"#).unwrap();

    let mut proc = PatProcess::spawn(opts).expect("spawn");
```

```rust
use std::io::{BufRead, BufReader};
use std::path::PathBuf;
use std::process::{Child, Command, Stdio};
use std::time::{Duration, Instant};

pub struct PatSpawnOptions {
    pub binary: PathBuf,
    pub config_path: PathBuf,
    pub mbox_dir: PathBuf,
    pub http_listen_port: u16, // 0 = ephemeral
    pub pid_file: PathBuf,
}

pub struct PatProcess {
    child: Option<Child>,
    pid_file: PathBuf,
    http_port: u16,
}

impl PatProcess {
    /// Spawn Pat. Blocks until Pat's HTTP server has announced its listening
    /// port on stdout (Pat prints "Listening on 127.0.0.1:PORT" when --listen
    /// resolves). Returns after the port is known. Caller is responsible for
    /// ensuring config_path exists and is valid.
    pub fn spawn(opts: PatSpawnOptions) -> std::io::Result<Self> {
        std::fs::create_dir_all(&opts.mbox_dir)?;
        if let Some(parent) = opts.pid_file.parent() {
            std::fs::create_dir_all(parent)?;
        }
        let listen = format!("127.0.0.1:{}", opts.http_listen_port);
        let mut cmd = Command::new(&opts.binary);
        cmd.arg("http")
            .arg("--listen").arg(&listen)
            .arg("--mbox").arg(&opts.mbox_dir)
            .arg("--config").arg(&opts.config_path)
            .stdout(Stdio::piped())
            .stderr(Stdio::piped());
        let mut child = cmd.spawn()?;

        // Parse Pat's stdout for the listen line. Pat v1.0.0 prints
        // "HTTP server listening on 127.0.0.1:53421" or similar. We match
        // the "127.0.0.1:" substring and extract the port.
        let stdout = child.stdout.take().expect("piped stdout");
        let reader = BufReader::new(stdout);
        let mut http_port: u16 = 0;
        let deadline = Instant::now() + Duration::from_secs(10);
        for line in reader.lines() {
            if Instant::now() > deadline {
                break;
            }
            let line = match line {
                Ok(l) => l,
                Err(_) => continue,
            };
            if let Some(idx) = line.find("127.0.0.1:") {
                let tail = &line[idx + "127.0.0.1:".len()..];
                let port_str: String = tail.chars().take_while(|c| c.is_ascii_digit()).collect();
                if let Ok(p) = port_str.parse::<u16>() {
                    http_port = p;
                    break;
                }
            }
        }
        if http_port == 0 {
            // Give up; Pat did not announce. Kill it and error.
            let _ = child.kill();
            return Err(std::io::Error::new(
                std::io::ErrorKind::TimedOut,
                "pat did not announce HTTP listen port within 10s",
            ));
        }

        // Write PID file.
        std::fs::write(&opts.pid_file, child.id().to_string())?;

        Ok(PatProcess {
            child: Some(child),
            pid_file: opts.pid_file,
            http_port,
        })
    }

    pub fn http_port(&self) -> u16 {
        self.http_port
    }

    pub fn is_running(&mut self) -> bool {
        if let Some(child) = self.child.as_mut() {
            match child.try_wait() {
                Ok(None) => true,
                Ok(Some(_)) => false,
                Err(_) => false,
            }
        } else {
            false
        }
    }

    pub fn shutdown(&mut self, timeout: Duration) -> std::io::Result<()> {
        if let Some(mut child) = self.child.take() {
            use nix::sys::signal::{kill, Signal};
            use nix::unistd::Pid;
            let pid = Pid::from_raw(child.id() as i32);
            let _ = kill(pid, Signal::SIGTERM);

            let deadline = Instant::now() + timeout;
            loop {
                match child.try_wait() {
                    Ok(Some(_)) => break,
                    Ok(None) => {
                        if Instant::now() > deadline {
                            let _ = child.kill();
                            let _ = child.wait();
                            break;
                        }
                        std::thread::sleep(Duration::from_millis(100));
                    }
                    Err(e) => return Err(e),
                }
            }
        }
        let _ = std::fs::remove_file(&self.pid_file);
        Ok(())
    }
}

impl Drop for PatProcess {
    fn drop(&mut self) {
        if self.child.is_some() {
            // Best-effort SIGKILL; process must not outlive the struct.
            if let Some(mut child) = self.child.take() {
                let _ = child.kill();
                let _ = child.wait();
            }
            let _ = std::fs::remove_file(&self.pid_file);
        }
    }
}
```

```rust
use serde::Deserialize;

pub enum MailboxFolder { Inbox, Sent, Outbox, Archive }

impl MailboxFolder {
    fn as_path(&self) -> &'static str {
        match self {
            MailboxFolder::Inbox => "in",
            MailboxFolder::Sent => "sent",
            MailboxFolder::Outbox => "out",
            MailboxFolder::Archive => "archive",
        }
    }
}

#[derive(Debug, Clone)]
pub struct Message {
    pub mid: String,
    pub subject: String,
    pub from: String,
    pub date: String,
    pub unread: bool,
    pub body_size: u64,
}

#[derive(Debug, Deserialize)]
struct PatMessageDto {
    #[serde(rename = "MID")] mid: String,
    #[serde(rename = "Subject")] subject: String,
    #[serde(rename = "From")] from: PatAddr,
    #[serde(rename = "Date")] date: String,
    #[serde(rename = "Unread", default)] unread: bool,
    #[serde(rename = "BodySize", default)] body_size: u64,
}

#[derive(Debug, Deserialize)]
struct PatAddr { #[serde(rename = "Addr")] addr: String }

impl From<PatMessageDto> for Message {
    fn from(d: PatMessageDto) -> Self {
        Message {
            mid: d.mid, subject: d.subject, from: d.from.addr,
            date: d.date, unread: d.unread, body_size: d.body_size,
        }
    }
}

#[derive(Debug)]
pub enum PatClientError {
    Http(reqwest::Error),
    Status(u16),
}

impl std::fmt::Display for PatClientError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            PatClientError::Http(e) => write!(f, "HTTP error: {}", e),
            PatClientError::Status(s) => write!(f, "Pat returned status {}", s),
        }
    }
}
impl std::error::Error for PatClientError {}

pub struct PatClient {
    base_url: String,
    http: reqwest::blocking::Client,
}

impl PatClient {
    pub fn new(base_url: impl Into<String>) -> Self {
        let http = reqwest::blocking::Client::builder()
            .timeout(std::time::Duration::from_secs(5))
            .build().expect("reqwest build");
        PatClient { base_url: base_url.into(), http }
    }

    pub fn list(&self, folder: MailboxFolder) -> Result<Vec<Message>, PatClientError> {
        let url = format!("{}/api/mailbox/{}", self.base_url, folder.as_path());
        let resp = self.http.get(&url).send().map_err(PatClientError::Http)?;
        if !resp.status().is_success() {
            return Err(PatClientError::Status(resp.status().as_u16()));
        }
        let dtos: Vec<PatMessageDto> = resp.json().map_err(PatClientError::Http)?;
        Ok(dtos.into_iter().map(Message::from).collect())
    }

    pub fn send(&self, to: &[&str], subject: &str, body: &str) -> Result<String, PatClientError> {
        #[derive(serde::Serialize)]
        struct Out<'a> {
            #[serde(rename = "To")] to: Vec<Addr<'a>>,
            #[serde(rename = "Subject")] subject: &'a str,
            #[serde(rename = "Body")] body: &'a str,
        }
        #[derive(serde::Serialize)]
        struct Addr<'a> { #[serde(rename = "Addr")] addr: &'a str }
        #[derive(Deserialize)]
        struct Resp { #[serde(rename = "MID")] mid: String }

        let msg = Out {
            to: to.iter().map(|a| Addr { addr: a }).collect(),
            subject, body,
        };
        let url = format!("{}/api/mailbox/out", self.base_url);
        let resp = self.http.post(&url).json(&msg).send().map_err(PatClientError::Http)?;
        if !resp.status().is_success() {
            return Err(PatClientError::Status(resp.status().as_u16()));
        }
        let body: Resp = resp.json().map_err(PatClientError::Http)?;
        Ok(body.mid)
    }
}
```

```rust
use std::io::{BufRead, BufReader, Read, Write};

pub struct TransmissionPlan {
    pub target: String,
    pub session_count: u32,
    pub expected_duration_s: u32,
    pub content: String,
    pub freq_mode_band: String,
    pub callsign: String,
}

pub enum ConsentOutcome {
    Granted,
    Aborted,
}

pub fn check_consent<R: Read, W: Write>(plan: &TransmissionPlan, input: R, mut output: W) -> ConsentOutcome {
    let _ = writeln!(
        output,
        "\nWARNING: Live amateur radio transmission.\n\
         This tool will transmit on the amateur radio network under callsign {callsign}.\n\n\
         Planned activity:\n  \
         - Target: {target}\n  \
         - Session count: {count}\n  \
         - Expected duration: {duration} s\n  \
         - Transmission content: {content}\n  \
         - Frequency / mode / band: {fmb}\n\n\
         By typing \"go\" and pressing Enter, you confirm:\n  \
         - You are the station licensee (or their authorized deputy).\n  \
         - You accept responsibility under 47 CFR Part 97 for these transmissions.\n  \
         - You consent to this specific run only; no future run is authorized.\n  \
         - You will monitor for completion.\n\n\
         Type \"go\" to proceed, anything else to abort:\n> ",
        callsign = plan.callsign,
        target = plan.target,
        count = plan.session_count,
        duration = plan.expected_duration_s,
        content = plan.content,
        fmb = plan.freq_mode_band,
    );
    let _ = output.flush();

    let mut reader = BufReader::new(input);
    let mut line = String::new();
    if reader.read_line(&mut line).is_err() {
        return ConsentOutcome::Aborted;
    }
    // Strip exactly one trailing newline (Unix) or CRLF (Windows), nothing else.
    // "go" exactly; no surrounding whitespace, no case variation.
    let trimmed = line.strip_suffix('\n').unwrap_or(&line).strip_suffix('\r').unwrap_or(line.strip_suffix('\n').unwrap_or(&line));
    if trimmed == "go" {
        ConsentOutcome::Granted
    } else {
        let _ = writeln!(output, "Aborted — no transmission occurred.");
        ConsentOutcome::Aborted
    }
}
```

```rust
fn run_smoke(callsign: &str, password: &str, grid: &str, plan: &TransmissionPlan) -> Result<u32, String> {
    let home = std::env::var_os("HOME").ok_or("HOME unset")?;
    let pat_config_path = PathBuf::from(&home).join(".config").join("pat").join("config.json");
    write_pat_config(
        &pat_config_path,
        &WizardInput {
            callsign: callsign.into(),
            password: password.into(),
            grid_square: grid.into(),
            mbo_address: format!("{}@winlink.org", callsign),
        },
    ).map_err(|e| format!("write pat config: {}", e))?;

    let mbox_dir = PathBuf::from(&home).join(".local").join("share").join("tuxlink").join("mbox");
    let pid_file = PathBuf::from(&home).join(".local").join("state").join("tuxlink").join("pat.pid");
    let opts = PatSpawnOptions {
        binary: PathBuf::from("pat"),
        config_path: pat_config_path,
        mbox_dir,
        http_listen_port: 0,
        pid_file,
    };
    let mut proc = PatProcess::spawn(opts).map_err(|e| format!("spawn pat: {}", e))?;
    let client = PatClient::new(format!("http://127.0.0.1:{}", proc.http_port()));

    let _mid = client
        .send(&[&plan.target], "tuxlink setup test", &plan.content)
        .map_err(|e| format!("send to outbox: {}", e))?;

    let connect_url = format!("http://127.0.0.1:{}/api/connect?url=telnet", proc.http_port());
    reqwest::blocking::Client::new()
        .post(&connect_url)
        .timeout(Duration::from_secs(30))
        .send()
        .map_err(|e| format!("trigger connect: {}", e))?;

    let deadline = Instant::now() + Duration::from_secs(30);
    loop {
        let msgs = client.list(MailboxFolder::Inbox).map_err(|e| format!("list inbox: {}", e))?;
        if msgs.iter().any(|m| m.from.contains("SERVICE@winlink.org")) {
            let _ = proc.shutdown(Duration::from_secs(5));
            return Ok(1);
        }
        if Instant::now() > deadline {

            return Err("no reply from SERVICE@winlink.org within 30s".into());
        }

    }
```

```rust
use tauri::tray::{TrayIconBuilder, TrayIconEvent};

    vec!["tray:show_hide", "tray:new_message", "tray:quit"]

pub fn install<R: Runtime>(app: &AppHandle<R>) -> tauri::Result<()> {
    let show_hide = MenuItemBuilder::with_id("tray:show_hide", "Show/Hide Window").build(app)?;
    let new_msg = MenuItemBuilder::with_id("tray:new_message", "New Message").build(app)?;
    let quit = MenuItemBuilder::with_id("tray:quit", "Quit").build(app)?;
    let menu = MenuBuilder::new(app)
        .items(&[&show_hide, &new_msg, &quit])
        .build()?;
    let app_clone = app.clone();
    let _tray = TrayIconBuilder::new()
        .menu(&menu)
        .tooltip("Tuxlink")
        .on_menu_event(move |app, event| {
            let id = event.id().as_ref();
            match id {
                "tray:show_hide" => {
                    if let Some(window) = app.get_webview_window("main") {
                        if window.is_visible().unwrap_or(false) {
                            let _ = window.hide();
                        } else {
                            let _ = window.show();
                            let _ = window.set_focus();
                        }
                    }
                }
                "tray:new_message" => {
                    if let Some(window) = app.get_webview_window("main") {
                        let _ = window.show();
                        let _ = window.set_focus();
                    }
                    let _ = app.emit("menu", "menu:file:new");
                }
                "tray:quit" => {
                    app.exit(0);
                }
                _ => {}
            }
        })
        .on_tray_icon_event(move |tray, event| {
            if let TrayIconEvent::Click { .. } = event {
                if let Some(window) = app_clone.get_webview_window("main") {
                    let _ = window.show();
                    let _ = window.set_focus();
                }
            }
        })
        .build(app)?;
    Ok(())
}
```

```ts
export type WizardStep = 'account' | 'credentials' | 'test_send' | 'complete';

export interface WizardState {
  step: WizardStep;
  hasAccount: boolean | null;
  callsign: string;
  password: string;
  gridSquare: string;
  mboAddress: string;
  testSendStatus: 'idle' | 'connecting' | 'sending' | 'waiting_reply' | 'success' | { error: string };
}

export const initialWizardState = (): WizardState => ({
  step: 'account',
  hasAccount: null,
  callsign: '',
  password: '',
  gridSquare: '',
  mboAddress: '',
  testSendStatus: 'idle',
});

export function advanceWizard(s: WizardState): WizardState {
  if (s.step === 'account') {
    if (s.hasAccount === null) {
      throw new Error('Cannot advance from account step without hasAccount decision');
    }
    return { ...s, step: 'credentials' };
  }
  if (s.step === 'credentials') {
    if (!s.callsign || !s.password || !s.gridSquare) {
      throw new Error('Cannot advance from credentials step with missing fields');
    }
    return { ...s, step: 'test_send' };
  }
  if (s.step === 'test_send') {
    if (s.testSendStatus !== 'success') {
      throw new Error('Cannot advance from test_send without success');
    }
    return { ...s, step: 'complete' };
  }
  return s;
```

```ts
const CALLSIGN_RE = /^[A-Z0-9]{3,6}$/;
const GRID4_RE = /^[A-R]{2}[0-9]{2}$/;
const GRID6_RE = /^[A-R]{2}[0-9]{2}[a-x]{2}$/;

export function normalizeCallsign(c: string): string {
  return c.trim().toUpperCase();
}

export function validateCallsign(c: string): string | null {
  const n = normalizeCallsign(c);
  if (!CALLSIGN_RE.test(n)) {
    return 'Callsign must be 3-6 letters/digits (e.g. W4PHS).';
  }
  return null;
}

export function normalizeGrid(g: string): string {
  const t = g.trim();
  if (t.length === 4) return t.toUpperCase();
  if (t.length === 6) return t.slice(0, 4).toUpperCase() + t.slice(4).toLowerCase();
  return t;
}

export function validateGrid(g: string): string | null {
  const n = normalizeGrid(g);
  if (GRID4_RE.test(n) || GRID6_RE.test(n)) return null;
  return 'Grid square must be 4 or 6 chars: 2 letters A-R, 2 digits, optional 2 letters a-x (e.g. EM75 or EM75xx).';
}
```

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WizardInput {

struct PatConfig<'a> {
    mycall: &'a str,
    secure_login_password: &'a str,
    locator: &'a str,
    mbo_address: &'a str,
}

pub fn write_pat_config(path: &Path, input: &WizardInput) -> std::io::Result<()> {
    if let Some(parent) = path.parent() { std::fs::create_dir_all(parent)?; }
    let cfg = PatConfig {
        mycall: &input.callsign,
        secure_login_password: &input.password,
        locator: &input.grid_square,
        mbo_address: &input.mbo_address,
    };
    let json = serde_json::to_string_pretty(&cfg).expect("serialize");
    std::fs::write(path, json)
}

#[tauri::command]
pub async fn wizard_complete(input: WizardInput) -> Result<(), String> {
    // 1. Write Pat config.

    // 2. Write tuxlink config (schema from Task 2).
    let tx_config_path = crate::config::config_path();
    if let Some(parent) = tx_config_path.parent() { std::fs::create_dir_all(parent).map_err(|e| e.to_string())?; }
    let tx_config = crate::config::Config {

        pat_mbo_address: input.mbo_address.clone(),
        winlink_password_present: true,
        wizard_completed: true,
    };
    let json = serde_json::to_string_pretty(&tx_config).map_err(|e| e.to_string())?;
    std::fs::write(&tx_config_path, json).map_err(|e| e.to_string())?;
    Ok(())
}
```

```rust
#[derive(Debug, Clone, Deserialize)]
pub struct FullMessage {
    #[serde(rename = "MID")] pub mid: String,
    #[serde(rename = "Subject")] pub subject: String,
    #[serde(rename = "From")] #[serde(deserialize_with = "deser_addr_single")] pub from: String,
    #[serde(rename = "To", default, deserialize_with = "deser_addr_list")] pub to: Vec<String>,
    #[serde(rename = "Date")] pub date: String,
    #[serde(rename = "Body", default)] pub body: String,
}

fn deser_addr_single<'de, D: serde::Deserializer<'de>>(d: D) -> Result<String, D::Error> {
    let a = PatAddr::deserialize(d)?;
    Ok(a.addr)
}
fn deser_addr_list<'de, D: serde::Deserializer<'de>>(d: D) -> Result<Vec<String>, D::Error> {
    let v: Vec<PatAddr> = Vec::deserialize(d)?;
    Ok(v.into_iter().map(|a| a.addr).collect())
}

impl PatClient {
    pub fn read(&self, mid: &str) -> Result<FullMessage, PatClientError> {
        let url = format!("{}/api/mailbox/in/{}", self.base_url, mid);
        let resp = self.http.get(&url).send().map_err(PatClientError::Http)?;
        if !resp.status().is_success() {
            return Err(PatClientError::Status(resp.status().as_u16()));
        }
        resp.json().map_err(PatClientError::Http)
    }
}
```

```ts
// src/compose/useDraft.ts
export interface Draft {
  id: string;
  to: string;
  subject: string;
  body: string;
}

const PREFIX = 'tuxlink.draft.';

export function saveDraft(d: Draft): void {
  localStorage.setItem(PREFIX + d.id, JSON.stringify(d));
}

export function loadDraft(id: string): Draft | null {
  const raw = localStorage.getItem(PREFIX + id);
  if (!raw) return null;
  try { return JSON.parse(raw) as Draft; } catch { return null; }
}

export function clearDraft(id: string): void {
  localStorage.removeItem(PREFIX + id);
}

export function listDraftIds(): string[] {
  const ids: string[] = [];
  for (let i = 0; i < localStorage.length; i++) {
    const k = localStorage.key(i);
    if (k && k.startsWith(PREFIX)) ids.push(k.slice(PREFIX.length));
  }
  return ids;
}
```

```tsx
// src/compose/Compose.tsx
import * as Dialog from '@radix-ui/react-dialog';
import { useEffect, useRef, useState } from 'react';
import { invoke } from '@tauri-apps/api/core';
import { saveDraft, loadDraft, clearDraft, type Draft } from './useDraft';

const AUTOSAVE_MS = 2000;

export function Compose({ open, onOpenChange, draftId }: {
  open: boolean; onOpenChange: (o: boolean) => void; draftId: string;
}) {
  const [draft, setDraft] = useState<Draft>(
    () => loadDraft(draftId) ?? { id: draftId, to: '', subject: '', body: '' }
  );
  const [sending, setSending] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const autosaveRef = useRef<number>();

  useEffect(() => {
    if (autosaveRef.current) window.clearTimeout(autosaveRef.current);
    autosaveRef.current = window.setTimeout(() => saveDraft(draft), AUTOSAVE_MS);
    return () => { if (autosaveRef.current) window.clearTimeout(autosaveRef.current); };
  }, [draft]);

  async function doSend() {
    setSending(true); setError(null);
    try {
      const recipients = draft.to.split(',').map((s) => s.trim()).filter(Boolean);
      await invoke('message_send', { to: recipients, subject: draft.subject, body: draft.body });
      clearDraft(draft.id);
      onOpenChange(false);
    } catch (e) {
      setError(String(e));
    } finally {
      setSending(false);
    }
  }

  function doSaveAndClose() {
    saveDraft(draft);
    onOpenChange(false);
  }

  function onKeyDown(e: React.KeyboardEvent) {
    if ((e.ctrlKey || e.metaKey) && e.key === 'Enter') { e.preventDefault(); doSend(); }
    if ((e.ctrlKey || e.metaKey) && e.key === 's') { e.preventDefault(); doSaveAndClose(); }
```

```rust
use std::collections::VecDeque;

pub struct LogRing {
    capacity: usize,
    entries: VecDeque<String>,
}

impl LogRing {
    pub fn with_capacity(capacity: usize) -> Self {
        Self { capacity, entries: VecDeque::with_capacity(capacity) }
    }

    pub fn push(&mut self, line: String) {
        if self.entries.len() == self.capacity { self.entries.pop_front(); }
        self.entries.push_back(line);
    }

    pub fn snapshot(&self) -> Vec<String> {
        self.entries.iter().cloned().collect()
    }
}
```

```rust
pub fn bundled_pat_path() -> std::path::PathBuf {
    // Tauri's externalBin copies the binary next to the app binary, suffixed
    // with the target triple. Try the triple-suffixed name first, fall back
    // to plain "pat" (for dev), finally fall back to PATH.
    let exe = std::env::current_exe().ok();
    if let Some(dir) = exe.and_then(|p| p.parent().map(|d| d.to_path_buf())) {
        for candidate in &["pat", "pat-x86_64-unknown-linux-gnu"] {
            let p = dir.join(candidate);
            if p.is_file() { return p; }
        }
    }
    std::path::PathBuf::from("pat")
}
```

```bash
# Run cargo test and vitest, capture stderr, fail if any forbidden noise
```

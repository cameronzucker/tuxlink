# Code of Conduct

The Tuxlink project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), a widely-adopted standard for inclusive community conduct in open-source projects.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Conduct concerns should be reported privately to the project maintainer at <conduct@tuxlink.org>. Reports are reviewed promptly and kept confidential to the extent possible while resolving the concern. The reporter is informed of the outcome.

## Scope

This Code of Conduct applies to all project spaces — the GitHub repository (issues, pull requests, discussions, code review), any associated community channels, and public events where someone is representing the project.

## Enforcement

The maintainer is responsible for enforcement and may take any action they deem appropriate, including warning, temporary ban, or permanent ban from project spaces. Enforcement decisions follow the Contributor Covenant 2.1 enforcement guidelines linked above.
